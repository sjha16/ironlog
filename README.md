# IronLog

IronLog is a modern workout tracking app built using React, Capacitor, and Android Studio.

The app is designed to provide a clean and focused workout logging experience with:

- Workout session tracking
- Exercise history
- Progress visualization
- Custom exercises
- Backup & restore support
- Multiple exercise log types
- Calendar-based workout tracking

## Features

### Workout Tracking

- Start and end workout sessions
- Automatic workout duration tracking
- Day-wise workout history

### Exercise Logging

Supports multiple logging styles:

- Weight + reps
- Reps only
- Timer-based exercises
- Weight + time exercises

### Progress System

- Exercise-wise progress tracking
- Personal best tracking
- Session history
- Interactive progress graphs

### Workout Library

- Organized muscle-group exercise system
- Custom exercise creation
- Expandable exercise categories

### Data Management

- Local backup & restore
- Offline-first design
- Persistent local storage

## Tech Stack

- React
- JavaScript
- Capacitor
- Android Studio
- LocalStorage
